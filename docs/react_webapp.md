# React Frontend

This frontend is built with a react framework and FastAPI backend. 

## How to Contribute 

### First Time Setup

- Prerequistes:
    - npm
    - git
    - python (3.x.x)
    - TODO: update

To install node dependencies, run:
```bash
cd web_app/auv_gui
npm install
```

To startup the GUI:
```bashs
sh startup.sh
```
(Depending on what python version you have, you may have to edit startup.sh to be python or python3.)
